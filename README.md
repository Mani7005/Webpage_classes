# 📚 Ignited Minds Tuition Centre Website

A clean, responsive single-page website designed for **Ignited Minds Tuition Centre** to establish an online presence and provide students and parents with essential information about the institute.

## 🌐 Live Demo

https://webpage-classes.vercel.app/

## ✨ Features

- Responsive single-page design
- Modern and intuitive user interface
- Hero section with image slider
- About section
- Subjects offered
- Student achievements and results
- Contact information with Google Maps integration
- Smooth scrolling navigation
- Mobile-friendly layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/repository-name.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## 🎯 Purpose

This project was built to create a simple and professional website for a neighbourhood tuition centre, making it easy for students and parents to learn about the centre, explore the subjects offered, view student achievements, and get in touch.
